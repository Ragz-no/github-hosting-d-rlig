installer Heroku CLI

Last ned og installer Heroku CLI.

Hvis du ikke har lagt inn ssh nøkkel ennå, gå inn på brukeren din og gjør dette.

$ heroku login

Lag et nytt git repository

Initialiser git repository i en ny eller gammel mappe

$ cd my-project/
  $ git init
  $ heroku git:remote -a heroku-vs-git

Host nettsiden din

push koden din og deploy med heroku gjennom Git.

$ git add .
  $ git commit -am "make it better"
  $ git push heroku master

Eksisterende git repository

For eksisterende repositories, legg kun til heroku remote

$ heroku git:remote -a heroku-vs-git
